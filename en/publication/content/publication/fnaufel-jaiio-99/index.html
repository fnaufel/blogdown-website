C++ Idioms for Concurrent Operations

Fernando Náufel do Amaral, P. G. Rabelo, S. E. R. Carvalho
1999-01-01
PDF

Abstract

Well-integrated development tools, allowing automatic code generation from visual representations of analysis and design decisions, are important assets in handling the complexities of today’s software. This paper describes several message passing semantics for the expression of concurrency in a new object-oriented visual development system, along with the C$^++$ idioms generated for asynchronous messages.
